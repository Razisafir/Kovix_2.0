# KOVIX User Journey

1. **Create Project** — Open Command Palette → "KOVIX: New Project" → 4-step wizard (name, description, tech stack, goals)
2. **Refine Idea** — 3-5 conversational turns with the AI agent to clarify scope, constraints, and success criteria
3. **Review Plan** — AI generates a plan with checkboxes per step and milestone markers; user can include/exclude steps
4. **Choose Stop Mode** — Select autonomy level: Every Milestone / Major Milestones / Selective / Full Auto
5. **Execute** — Agent reads files, writes code, runs commands — pauses at milestones based on stop mode
6. **Memory** — Learnings auto-extracted to universal cross-project memory store after milestone completions
7. **Resume** — Pick up exactly where you left off after any IDE restart; paused executions offer resume prompt
